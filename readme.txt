site2site copies any content from one site to another in a Wordpress Multisite installation.

== Description ==

With site2site you can copy any post, page or custom post type from one site to another, in a multisite installation. You can choose which post type you want to copy, and whether to keep the author of the original content or choose a new one.

== Installation ==

1. Upload site2site to the `/wp-content/plugins/` directory
2. Go to the network admin and activate the plugin through the 'Plugins' menu in WordPress
3. In the settings menu you will find a new option called `Copy content to another site`
